Computer Science & B.I.T

This is a brief write-up about the Computer Science & B.I.T Departments in the Canadian University of Modern Technology. These Departments offer a couple of similar modules but we shall only be looking at 2.

Namely;

  • Web Design & Development
  • Statistical Analysis

Web Development

This module is taught by Mr. Osman Kanu.

It is about the creation & modification of web pages.

Statistical Analysis

This module is taught by Mr. Alhassan Koroma.

It is about the analysis and manipulation of figures.
